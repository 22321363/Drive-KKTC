# Drive KKTC

Drive KKTC is a bilingual tourism website developed during a summer internship to promote driving tourism across the Turkish Republic of Northern Cyprus (TRNC).

The website provides carefully planned driving routes, destination highlights, practical travel information, downloadable itineraries, and audio guides to help visitors explore the island.

---

## Features

- Five themed driving routes across Northern Cyprus
- English and Turkish language support
- Responsive design for desktop and mobile devices
- Route-specific hero illustrations
- Downloadable PDF itineraries
- Audio guides for each route
- Practical travel information
- SEO-friendly structure and metadata
- Custom 404 page

## Responsive Design

The website is optimized for desktops, tablets, and mobile devices. Custom mobile hero SVG illustrations are used to preserve visual balance on smaller screens.
