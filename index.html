About the project

Why this exists, and how it's built.

The idea

Drive KKTC was built as an intern project for the 2026 program, inspired by Drive Japan: a dedicated page for every experience, instead of scattered blog posts and outdated forum threads. Nothing like this existed for Northern Cyprus, so we built it — five self-researched routes, each shipped as a real page with a real URL.

How it's built

Plain HTML, CSS and JavaScript — no framework. Each route page includes an interactive Leaflet map, stop-by-stop cards with photos, practical info, an insider tip (with audio), and a save-for-later system that works entirely in your browser.
